El primer capítulo nos habla  sobre el funcionamiento básico de un sitio web, nos indica que un sitio web, son varias páginas 
web, enlazadas(por hipervínculos y bajo un mismo dominio) y están almacenadas en un servidor el cual tiene la función de proveer de dichas páginas al 
usuario las 24 horas del dia.


Las páginas están conformadas por HTML, CSS Y JS, se utilizan diferentes protocolos para acceder a la página, por ejemplo: http y https. También se 
utilizan lenguajes de servidor, para trabajar con el contenido dinámico. Se puede utilizar un conjunto de tecnologías para levantar un servidor
local, por ejemplo: mamp.


Hay que pagar por los dominios(es un nombre más fácil de recordar, así el usuario no debe memorizar una ip, los nombres de dominio se almacenan en 
servidores llamados; DNS) y de estos se pueden crear subdominios para tener más páginas, los archivos se pueden subir a un servidor con el protocolo FTP y por 
medio de cpanel, se pueden administrar el servidor de una manera gráfica.
